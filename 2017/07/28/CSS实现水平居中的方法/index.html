CSS实现水平居中的方法

发表于 2017-07-28 | 分类于 css

摘要:
用CSS实现水平居中的几种方法，以及各自的优缺点。

传统方法:

第一种方法

已知宽度情况，设置容器的margin左右为auto。

第二种方法

已知宽度情况，设置容器的position为absolute，然后设置left为50%，接着设置margin-left为自身宽度的一半。
以上两种方法的优点：实现方法简单易懂，浏览器兼容性强；缺点：扩展性差，无法自适应未知项情况。

inline-block实现水平居中

父类容器设置text-align的属性为center，子类容器设置display: inline-block;这样就可以实现居中，但是要处理inline-block的兼容性问题。
解决由回车符带来的空白间距：zoom:1; letter-spacing: normal; word-spacing: normal;。
优点：简单易懂，扩展性强； 缺点：需要额外处理inline-block的浏览器兼容性。

浮动实现水平居中

设置父类容器float为left，然后position为relative，设置left为50%，然后子类容器设置float为left;然后position为relative，然后位置设置成父类相反的方向为50%。
优点：兼容性强，扩展性强； 缺点：实现原理较复杂。

绝对定位水平居中

设置第一层容器position为relative,然后第二层容器的 position为absolute，接着设置left为50%；第三层内容容器设置float为left，接着position为relative，然后设置right为50%。
优点：扩展性强，兼容性强; 缺点：理解性难。

CSS3flex水平居中布局：

父类设置display: box; box-orient: horizontal; box-pack: center;
优点：实现便捷，扩展性强 缺点：兼容性差。

CSS3fit-content水平居中布局

父类设置width: fit-content; margin-left: auto; margin-right: auto;
优点：简单易懂，扩展性强; 缺点：浏览器兼容性差
