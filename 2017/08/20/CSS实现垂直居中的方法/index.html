CSS实现垂直居中的方法

发表于 2017-08-20 | 分类于 css

摘要:
CSS实现垂直居中的方法。

传统方法

给元素设置高度后，设置position为absolute，top为50%，然后设置margin-top为负的自身高度的一半；也可以这样设置position: absolute; top: calc(50% - 自身高度的一半);calc(四则运算)动态计算长度值

transform方法

容器设置position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);
因为transform中的translate()使用百分比，让元素移动是相对于自身的宽度和高度来定的。但是会出现顶部被裁掉的情况。所以需要使用hack手段，设置transform-style:preserve-3d就可以了

flex方法：

第一种：

容器设置display: flex; align-items: center; justify-content: center;

第二种：

容器设置display: flex; flex-direction: column; justify-content: center;

绝对定位结合margin: auto方法

父类设置position: relative;子类设置position: absolute; top: 0; bottom: 0; margin: auto;
